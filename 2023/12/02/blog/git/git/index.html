git在vscode的使用

2023/12/2

1.下载git，windows的安装时直接一路都next

https://git-scm.com/download/win

2.打开vscode

进入设置

搜索设置

输入git安装目录下cmd里的git.exe的地址

重启vscode

ctrl+shift+P 输入git，进行初始化、连接远程仓库之类操作

配置SSL证书

2023/12/2

1.配置本地仓库的账户邮箱

git config --global user.name ‘用户名字’
git config --global user.email ‘邮箱地址’

2.打开git cmd，输入

ssh-keygen -t rsa -C ‘邮箱地址’

passphrase可以空白，为无限期
生成证书

3.在GitHub设置ssh

点击头像
进入Settings -> SSH and GPG keys -> New SSH key
title随意
打开C:\Users\用户名\.ssh（如果生成证书时修改了地址，就去你指定的地址）
里面有id_rsa.pub和id_rsa
key框里输入id_rsa.pub里的内容（用记事本打开id_rsa.pub即可）
点击Add SSH key

4.输入命令，path为证书路径

git config --global http.sslcainfo "Path"

5.验证

打开git bash输入

ssh -T git@github.com

输入yes回车，如果提示successfully之类字样则SSH连接正常

git pull <url>

git push
